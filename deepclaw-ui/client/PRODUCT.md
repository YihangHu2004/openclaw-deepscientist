# Product

## Register

product

## Users

AI 研究者和开发者，使用 DeepClaw 管理长期研究项目、与 AI agent 对话、浏览工作区文件。开源插件，面向技术社区（研究员、工程师）。典型使用场景：深夜盯着实验结果，多窗口工作流，注意力高度集中。

## Product Purpose

DeepClaw UI 是 OpenClaw/DeepScientist 系统的前端界面，让用户通过浏览器与 AI research agent 实时对话，管理研究项目，浏览和预览工作区文件输出。设计服务于工作流，不能抢夺注意力。

## Brand Personality

学术 · 赛博 · 专业。克制的技术感，不是喧嚣的科幻感。像一台精密仪器，不像游戏界面。

## Anti-references

- 典型 SaaS 奶油白（Notion、Coda 的 beige 调）
- Indie hacker 极简风（Linear 的 stone gray + purple）
- 霓虹赛博朋克（过亮的霓虹绿/粉，过度发光）
- AI 感审美：Inter 字体 + 紫蓝渐变 + 无限嵌套卡片
- 花哨动效（bounce、elastic、无意义的 hover 爆炸）

## Design Principles

1. **仪器精度**：每个元素都有功能，装饰必须服务于信息层级
2. **深度而非广度**：暗色背景不是风格，是让内容浮现的容器
3. **动效有意义**：过渡传递状态变化，不炫耀技术
4. **专家默认**：面向技术用户，不需要过度解释，密度优于空旷
5. **开源可信**：清晰、可预测、无黑盒感

## Accessibility & Inclusion

- 深色主题为主（使用场景多为低光环境）
- 文字对比度达到 WCAG AA
- 动效尊重 prefers-reduced-motion
